## TASK : สร้าง procedure หาสมาชิกใน list

- ถ้าใน list มีสมาชิกที่ต้องการหาให้คืนค่าเป็น true
- ถ้าใน list ไม่มีสมาชิกที่ต้องการหาในคืนค่าเป็น false
- FN Prototype : list, target => boolean

#### example

```
(procedure-name (list 1 2 3) 3)
> #t

(procedure-name (list 1 2 3) 5)
> #f

(procedure-name (list ) 3)
> #f



```
