- ex 0, 1, 1, 2, 3, 5, 8, 13, 21,...

- FN Prototype : number(position) => number(value)

#### example

```
(procedure-name 0)
> 0

(procedure-name 2)
> 1

(procedure-name 5)
> 5

(procedure-name 7)

```
